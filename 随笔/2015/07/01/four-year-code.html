四年大学，四年程序

2015-07-01

回头一看，我的大学也就这样过去了。大学四年我很贪心地接触了很多编程语言。回想起来，再也没有那种精力与勇气了。

大一刚上第一节 C 语言课程的时候，同学都是一样的表情。这是啥玩意，这就是编程！那时的我对编程没什么概念，也没有什么兴趣，乐于体力兼职赚点小钱，就连基础的 C++ 都差点挂了。大一最讨厌的事情就是每周五下午四节课的编程实验课，晴朗的天气，刺眼的阳光，清晰的蝉鸣，实验室里却是一片片同学们的敲击键盘声。整个大一下来，我连实验课的作业都是抄同学的，唯一庆幸的就是没有挂科。大一的我有两个人最敬佩。一个是同班的班长，他在我们还不知道什么叫做编程的时候就已经做了一个留言板的网站给我们玩耍，空间域名都搞定。第二个则是隔壁宿舍的一个同学，他对编程方面的领悟很高，一下子就抓住诀窍。在我第一节实验课不知所措的时候他已经把第七节实验课的都做完了，连老师都怂恿他加入自己的团队。可惜后来他放弃编程转去兼职卖楼了。

大二的我突然想通了，再这样下去月薪就真的是体力劳动的月薪了，于是辞职，上课，编程。那时的我 C 语言和 C++ 的编程能力实在不行，但是我有 Java 必修课，于是决定先把 Java 学好。Java 学起来没有C语言那么的吃力，那学期我的编程能力也提高了不少。寒假的时候碰巧从同学那里拿到一份 Java 的学习视频，才知道原来还有这玩意，于是整个寒假每天都会抽空待在电脑前看视频，跟着把例子实现出来。大二下学期回来后我跟着视频做出了一个坦克大战的桌面游戏，可惜后来无人问津被我抛弃了。大二下学期开始是我编程史上的转折点，那是开始学习做网站，虽然主要是学习 JSP，但我对前端比较感兴趣，在上面花的时间也很多，期末也写出了一个微博网站。大二的一门数据结构课程我也很认真听讲，实验课很认真地用 C 语言实现，虽然现在基本还给老师了，但算是对 C 语言的一个回顾。

大三开始，我开始勤工助学，负责的是学校的就业指导网站，用的是 ASP.NET，这是我大三一整年都有在接触的事情。大三的上学期在数据库课程还顺便学习了 PHP，期末大作业也用它写出了一个网站。大三下学期开始了，突然想接触一下移动开发，于是选定 Android 开发，但是在第一次安卓模拟器打开花了二十多分钟之后，我就放弃了它。后来接触到了 iOS 开发，并且在学校的苹果机房里玩了一会 Mac 电脑，对苹果的系统有一种探索新陆地的感觉，于是决定学习 iOS 开发。于是在大三寒假加入研究生师兄的项目开发 iOS 的项目开发，并发布了一款的 App 上苹果商店。大三的暑假里我开始寻找实习，找的是一个创业公司，由于我前端也学过，于是做的是前端。因为创业公司人手不太多，所以我也负责一些后端 Python 的开发，这次实习是我接触到前端众多知识的一个时期。

在大四上学期三个月的实习结束后，我迎来了接近半个月的清闲。这半个月里，我迷茫无助过，一遍复习以前学过的知识，一边开始找工作。终于在12月份签了一份 offer，然后又在1月份找了另一个家公司的实习，做的还是前端。与之前创业公司不同的是，在这公司我只做前端。在这段实习之前，实际上我没有独立做过正式的前端开发，后来也都慢慢克服困难习惯了。大四的下学期除了白天的前端实习之后，晚上我还要做一个 iOS 应用的毕业设计，那段时间即忙碌又充实。最后我迎来了毕业，确定了前端方向，并进了目前实习的公司。

是的，我接触了很多编程语言，但实际只是接触到了它们最基础的语法，做出来的项目也是最简单的。大学是让人来不断尝试的地方。不接触多一些东西，怎么能知道这是你想要的。每天接触到的东西都是感兴趣的才会有动力去完成工作。也许，大学四年的编程总结就是上面的几段话，但是其中的酸甜苦辣只有自己懂，最后我想对大学的我说一声，“辛苦了，接下来就放心交给我吧！”。

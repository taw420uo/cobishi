# cobish's blog

这是一个基于 ``github page`` 和 ``github api`` 搭建的单页面静态博客，点击[这里](http://cobish.github.io)查看博客。

## 写文章

在 ``Issues`` 使用 Markdown 书写文章，访问页面时即能够获取到 Issues 上的数据。

# cobish.github.io

## 环境变量

为了能够运行 ``Makefile`` 文件，需将 ``jekyll`` 和 ``git`` 设置为全局变量。Windows 请将 ``Makefile`` 修改成 ``Makefile.bat``。

jekyll 版本号

``` bash
$ jekyll -v
jekyll 2.5.3
```

## 分支

- ``source``分支用于开发，写文章；
- ``master``用于正式线上浏览。

## config 配置

- 将``百度统计``与``多说``的配置修改为自己的账号即可；
- 可根据自己的喜好对配置进行修改。

## terminal 命令

- 本地运行 ``jekyll serve``可进行调试；
- 运行 ``make`` 可进行提交代码。

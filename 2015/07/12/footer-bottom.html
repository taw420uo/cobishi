让footer固定在底部

Jul 12, 2015

角色介绍

  • class为main的一名战士：负责展示网页中的主体信息；
  • class为wrap的一名指挥官：是main的父亲；
  • class为footer的另一名战士：负责展示网页的版权信息等。

开头介绍

很久很久以前，footer喜欢跟在main的身后。当main的丰功伟绩很多的时候还好（即出现了滚动条）,但是当他的丰功伟绩屈指可数的时候（即页面内容较少时，撑不开浏览器的高度），他就受不了，风头都给footer抢走了一半。情况就是这样的：

<div class="main">这里是网页的主体</div>
<div class="footer">这里是页脚</div>

html, body{
    margin: 0;
    padding: 0;
}
.main {
    height: 300px;
    line-height: 300px;
    text-align: center;
    background-color: #ffffcc;
}
.footer {
    height: 30px;
    line-height: 30px;
    text-align: center;
    background-color: #ffcccc;
}

故事情节

前面说到main受不了了，于是他请了自己的父亲wrap帮忙，父亲用了一招min-height:100%，顺利地将footer驱逐到屏幕下面。但footer也不是吃素的，立马用了一招margin-top:-30px反击（它的高度为30px）。而这时父亲wrap发现自己没受伤，也发现main在丰功伟绩少的时候footer没跟在他后面，于是便就此作罢。


不久main才发现footer原来不安好心，当他的丰功伟绩很多时，footer总是抢占他的30px的功劳，于是他使出了必杀padding-bottom:30px，让出自己的30px位置给footer。


footer拿他们没辙了，事情也就此作罢，委屈地固定在了底部：

<div class="wrap">
    <div class="main">这里是网页的主体</div>
<div>
<div class="footer">这里是页脚</div>

html, body{
    margin: 0;
    padding: 0;
    height: 100%;
}
.wrap {
    overflow: hidden;
    min-height: 100%;
    height: 100%;
    height: auto;
}
.main {
    height: 300px;
    /*height: 800px;*/
    padding-bottom: 30px;
    line-height: 300px;
    text-align: center;
    background-color: #ffffcc;
}
.footer {
    margin-top: -30px;
    height: 30px;
    line-height: 30px;
    text-align: center;
    background-color: #ffcccc;
}
